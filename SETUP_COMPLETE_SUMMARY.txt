🔑 CRITICAL ENVIRONMENT VARIABLES:

You must add these 5 variables to Vercel:

1. DATABASE_URL
   └─ Neon PostgreSQL connection string
   └─ From: https://console.neon.tech

2. BLOB_READ_WRITE_TOKEN
   └─ Vercel Blob storage token
   └─ Auto-set (verify exists)

3. ADMIN_PASSWORD
   └─ For admin dashboard access
   └─ Choose secure password

4. NEXT_PUBLIC_SITE_URL
   └─ Your production URL
   └─ https://your-project.vercel.app

5. NEON_AUTH_COOKIE_SECRET
   └─ Authentication cookie secret
   └─ Generate: openssl rand -base64 32

See: VERCEL_COMPLETE_SETUP.md → Section 2 for details
